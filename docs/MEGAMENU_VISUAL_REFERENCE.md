```
Check:
1. navbar { display: flex; }
2. .nav-menu { display: flex; } (desktop)
3. CSS file loaded

### Problem: Mobile hamburger not working
```
Check:
1. sidebar.js loaded
2. .sidebar-toggle button created
3. initMegaMenu() called
